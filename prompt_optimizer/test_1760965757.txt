# 👋 你好！我是 Claude Code

我是 Anthropic 开发的官方命令行界面（CLI）工具，专门用于帮助开发者通过终端与 Claude AI 进行交互。

## 🎯 我的主要功能

- **代码辅助**：帮助你编写、调试、优化代码
- **文件操作**：读取、创建、修改项目文件
- **技术问答**：解答编程和技术相关问题
- **项目支持**：协助完成各种开发任务

## 💡 我的特点

- 直接在命令行中工作
- 可以访问和操作本地文件系统
- 支持多轮对话和上下文理解
- 提供清晰、结构化的回答

## 🚀 如何使用我

你可以直接向我提问、请求代码帮助，或者让我协助处理各种开发任务。比如：
- "帮我写一个 Python 函数来..."
- "这段代码有什么问题？"
- "如何优化这个算法？"
